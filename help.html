Adding notes

The program allows you to save all the words or expresions you discover to a list called Topic. You can add your own personal translations or use an online translator to do that automatically.



Please note: In the upper text field (1), enter text in foreign language. Below (2), enter text in the native language. Tip: Select a portion of text from any source, then click on the New button from panel (or "Add" from the indicator panel). If you want to add an image to a note use the screen clipping: Click on the button Image and then simply select any region that you want to capture. Other ways to add content are by selecting text blocks with the mouse to batch processing or try to get text from an image: Type "ocr" (Optical character recognition) or "i" in text field from New note dialog box, click on the button Add, then proceed in the same way to add an image.
